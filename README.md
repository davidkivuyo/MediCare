🏥 Medicare Portal (PHP)

📌 Overview

Medicare Portal is a healthcare web application developed using PHP and MySQL. It allows patients to book appointments, view doctor details, and manage their healthcare data efficiently. The system also includes an admin panel for managing doctors, appointments, and users.

---

🚀 Features

👤 User Features

- User Registration & Login (Authentication System)
- Book Appointments with Doctors
- View Doctor Details
- Manage Personal Information

🩺 Doctor Features

- View scheduled appointments
- Manage availability (if implemented)

🛠️ Admin Panel

- Add / Update / Delete Doctor Details
- Manage Appointments
- Manage Users
- Dashboard for system overview

---

🛠️ Tech Stack

- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL

⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/rashmiranjan-2005/Medicare-Portal-PHP.git

2. Move project to XAMPP "htdocs" folder

3. Start Apache and MySQL in XAMPP

4. Import database

- Open phpMyAdmin
- Create a database
- Import ".sql" file

5. Run the project

- Open browser

http://localhost/Medicare-Portal-PHP
